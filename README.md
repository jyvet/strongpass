StrongPass
=========

Bash script to generate a strong password of a given size.


Running StrongPass
-------------------

    Usage: spass.sh [OPTION]
      -n <num>    Number of characters (min:8, max:16)
      -h          Display help
      -c          Enable colors

Example
-------

    ./spass.sh -n 14

    Generating a 14 characters password:
    5C1Ml#OqFMVm3b
